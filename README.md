# 🚨 US Recession Early Warning System

## 📌 About This Project
An AI-powered Early Warning System that predicts 
US recession risk using real economic indicators 
and Machine Learning.

Built as part of my **Statistics × AI × Future World** 
series on LinkedIn.

## 🎯 What This Project Does
- Analyzes 7 years of real US economic data (2019–2026)
- Calculates a custom Recession Warning Score
- Predicts recession probability using Random Forest AI
- Visualizes unemployment, inflation, and GDP trends

## 🔍 Key Findings
- 🔴 2020 — Warning score hit 70 (COVID recession)
- 🟡 2022 — Warning score hit 60 (Inflation crisis)
- 🟢 2026 — Recession probability: 0.0% (Stable)

## 🛠 Tech Stack
- Python (pandas, numpy, matplotlib, seaborn)
- Scikit-learn (Random Forest Classifier)
- yfinance (S&P 500 real data)
- Google Colab
